# donationsapp

It is an application for people to support Jain organizations sitting in the comfort of their homes. 

Jain organizations like:
- Temples
- Gowshalas
- Stanaks
- Marasahab
... others

## Tech Stack
- Flutter
- Firebase Authentication
- Firestore Database
- Razorpay Integration
